# Sistema Web de Gestión de un Establecimiento de Comida
Sistema web de Gestión que permite administrar de manera sencilla los usuarios (clientes y empleados), mesas, reservaciones, promociones y zonas, además de generar reportes útiles para el análisis de la operación.

Para importar la base de datos, es necesario utilizar XAMPP y tener iniciados apache y mysql.
El comando para importar la base de datos es: mysql -u root -p < (ruta donde se encuentra la base de datos)

Ejemplo:
mysql -u root -p < "C:\xampp\htdocs\restaurante\database\schema.sql"

Y para la contraseña dejar en blanco y simplemente presionar la tecla "Enter"
